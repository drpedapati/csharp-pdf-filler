# Homebrew Tap Notes

This repository is not itself the tap.

Use this folder to prepare formula updates for a separate tap repository:
- copy [pdf-form-filler.rb.template](pdf-form-filler.rb.template) into the tap repo as `Formula/pdf-form-filler.rb`
- replace the placeholder version, homepage, URLs, and checksums
- point the formula at GitHub release tarballs generated from this repo

The release flow is documented in [docs/RELEASING.md](../../docs/RELEASING.md).
